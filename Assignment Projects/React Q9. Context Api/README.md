# Theme Switch App

## Project Details

A theme switch app using ReactJS context api. User can switch between light mode and dark mode.

## Tools

ReactJS

## How To Run Locally

Just copy all the file then run these two commands -

> To install all the dependencies -

```bash
npm i 
```

> To start the project -

```bash
npm run dev
```
